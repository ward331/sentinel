#### **2. Global Forest Watch Provider** ✅
- **File**: `internal/provider/globalforestwatch.go`
- **Source**: NASA VIIRS fire alerts via Global Forest Watch API
- **URL**: `https://data-api.globalforestwatch.org/dataset/nasa_viirs_fire_alerts/latest/query`
- **Features**:
  - Free, no API key required
  - 30-minute polling interval (1800 seconds)
  - Higher accuracy than NASA FIRMS alone
  - Fire detection metrics: brightness, FRP, confidence
  - Satellite: VIIRS (Visible Infrared Imaging Radiometer Suite)
  - Better detection of smaller fires and improved nighttime detection
- **Event Processing**:
  - Category: `wildfire`
  - Severity based on brightness and FRP
  - Badges: NASA VIIRS, High Confidence, Large Fire
  - Magnitude calculation from brightness and FRP

#### **3. Global Fishing Watch Provider** ✅
- **File**: `internal/provider/globalfishingwatch.go`
- **Source**: Global Fishing Watch vessel tracking API
- **URL**: `https://gateway.api.globalfishingwatch.org/v3/events`
- **Features**:
  - Free tier available (API key optional)
  - 1-hour polling interval (3600 seconds)
  - Vessel activity detection: fishing, transshipment, loitering, port visits
  - Dark vessel detection (flag = "UNK")
  - AIS-based tracking with event classification
- **Event Processing**:
  - Category: `maritime`
  - Severity: High for dark vessels and transshipment
  - Badges: Fishing, Dark Vessel, Transshipment
  - Risk assessment for illegal, unreported, unregulated (IUU) activity

    id INTEGER PRIMARY KEY AUTOINCREMENT,
    platform TEXT NOT NULL,          -- web, api, dataset, tool, rss, map
    category TEXT NOT NULL,          -- transport, conflict, aviation, maritime, etc.
    display_name TEXT NOT NULL,      -- Human-readable name
    profile_url TEXT NOT NULL,       -- URL to resource
    description TEXT,                -- Brief description
    credibility TEXT NOT NULL DEFAULT 'community', -- verified_osint, official, community
    is_builtin BOOLEAN NOT NULL DEFAULT 0, -- Pre-loaded in SENTINEL
    tags TEXT,                       -- JSON array of strings
    api_key_required BOOLEAN NOT NULL DEFAULT 0,
    free_tier BOOLEAN NOT NULL DEFAULT 1,
    notes TEXT
